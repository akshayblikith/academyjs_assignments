Section 2 - Basic HTML Elements

Technically, you can structure your content with just the div tag. But you should remember that your content is not only consumed by humans, but also by software tools like crawlers. Using tags to specify semantic information helps these tools understand your content better. For example, using keywords in heading tags is a known technique to improve your site's SEO.

We will learn about different HTML elements and their use cases in this chapter.

Block-Level and Inline-Level Elements

In HTML, elements are categorized as either block-level elements or inline-level elements.

A block-level element occupies 100% width of the parent element. Its height is equal to its content. Further, the browser adds a newline before and after the block. These are the default styles for a block-level element, but you can modify them using CSS.

Block-level elements can appear only inside the `body` element. You cannot use it inside the `head` element. Some examples of block-level elements include: div, code, blockquote, address, article, form, and so on.

An inline-level element on the other hand occupies both the width and the height equal to its content. It does not start a newline. Again, these are the default styles for an inline-level element, but you can modify them using CSS.

In general, you should put only inline-level elements or data in inline-level elements. You should not put block-level elements.

Paragraph

The `p` element creates a paragraph of text. It is a block-level element. You can add any inline-level elements like `span` and `a`. You cannot add block-level elements or nest paragraphs.

Line Break

The `br` element creates a line break. Can you guess if it is an inline-level element or a block-level element? You can use the shorthand syntax with this element.

Headings

There are six heading tags: h1, h2, h3, h4, h5, h6. By default, the h1 element is the biggest. The font size is decreased as the number increases. Similarly, the importance of the headings decreases from h1 to h6. In other words, the h1 element is given the highest importance, while the h6 element is given the least importance.

The organization of your headings is important for improving your site’s SEO.

Italics

The `i` element is used to mark text as idiomatic. The text is rendered in italics.

Bold

The `b` element is used to mark text to draw the reader’s attention. Before HTML4, this element was used to style text in boldface. However, you should use the font-weight CSS property for this.
